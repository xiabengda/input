# Less预处理语言
2024-05-28
## 介绍与安装使用
### 介绍
Less 是一种动态样式语言，一门css预处理语言，扩展了CSS语言，增加了变量、Mixin、函数等特性，使CSS更容易维护和扩展

预处理语言
- Sass
- Less
- Stylus
### 使用
在nodejs环境中使用less“
```bash
npm install -g less
```
浏览器中使用
```html
<link rel="stylesheet/less" type="text/css" href="styles.less" />  
<script src="https://cdn.jsdelivr.net/npm/less@4" ></script>
```
vscode中使用：安装easy less插件
